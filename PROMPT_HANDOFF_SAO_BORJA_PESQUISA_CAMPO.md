## 2. Delimitação obrigatória do projeto

Este handoff cobre apenas o projeto:

**São Borja — Pesquisa de Campo / Hub Logístico / Monetização de Ativos / Holding.**

Este projeto envolve a avaliação estratégica e operacional de oportunidades para monetização de ativos em São Borja/RS, especialmente galpão, terreno, oficina, loja comercial ou outros imóveis/estruturas com potencial econômico.

O foco é descobrir, validar e estruturar uma oportunidade de negócio real com base em dados de mercado, pesquisa presencial e validação de demanda.

Este handoff **não cobre** o projeto:

**São Borja Histórico-Cultural / Vídeo com IA / Episódio-piloto / Storytelling.**

Se encontrar materiais sobre São Borja histórico-cultural, não misturar com este handoff. Esse é outro projeto.

## 3. Tese estratégica atual

A tese atual do projeto não é:

> “Temos um galpão, então devemos criar imediatamente um centro logístico.”

A tese correta é:

> “Existe um ativo físico em São Borja que pode ser monetizado, mas antes de qualquer investimento relevante é necessário validar dores reais de mercado, identificar oportunidades com demanda, escolher um MVP enxuto e reduzir risco operacional e financeiro.”

O foco atual deve permanecer em:

- discovery;
- validação de mercado;
- pesquisa de campo;
- entrevistas;
- mapeamento de players;
- diagnóstico físico do ativo;
- seleção de MVP;
- estruturação de operação mínima;
- validação com possíveis clientes-piloto.
