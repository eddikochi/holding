# README — PACOTE PROMPT SÃO BORJA / HANDOFF IA

Este pacote contém arquivos auxiliares para solicitar e organizar o handoff do projeto:

**São Borja — Pesquisa de Campo / Hub Logístico / Holding / Monetização de Ativos.**

## Arquivos incluídos

1. `PROMPT_HANDOFF_SAO_BORJA_PESQUISA_CAMPO.md`
   - Prompt completo para pedir a geração do handoff específico deste projeto.

2. `MANIFESTO_ARQUIVOS_SAO_BORJA_HUB_LOGISTICO.md`
   - Lista dos arquivos encontrados na File Library e a função de cada um.

## Importante

Este pacote não contém os PDFs originais, pois eles foram localizados como arquivos da File Library e não como arquivos montados diretamente no ambiente sandbox.

Use este pacote junto com os PDFs originais, quando conseguir anexá-los ou abri-los na próxima IA.
